В настоящей реализции есть все необходимое для полноценной разработки на языке Java.

- Поддержка подсветки кода, автодополнение, поиск, рефакторинг, дебаг, тестирование.
- Так же есть поддержка фронтенда. Автодополнение html, css.
- Шаблоны freemarker обрабатываются так же как и html.

Для работы требуется neovim версии >= 0.11
